## Descrição do Projeto

Este projeto tem como objetivo criar uma criptografia e uma descriptografia de uma senha, proposto pelo professor Thaigo Ló, para a matéria Segurança e Auditoria de Sistemas. A aplicação foi desenvolvida utilizando HTML, CSS e JavaScript.
